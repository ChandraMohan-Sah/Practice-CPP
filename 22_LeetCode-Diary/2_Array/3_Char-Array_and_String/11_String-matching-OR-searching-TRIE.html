String Searching/Matching Using Trie
--------------------------------------------------------------------------------
Application
-Text Search:
-Plagiarism Detection:
-Spell Checkers:
-Finding occurrences of a pattern (substring).
--------------------------------------------------------------------------------
Belows algorithms are primarily used for string matching and string searching.
It is a popular algorithm for finding occurrences of a pattern (substring)
within a text (string) efficiently.

Algorithms :
1.Rabin-Karp Algorithm (Rolling Hash)
2.KNuth-Morris-Pratt (Prefix-Function)
3.Z-Function (Alternate to KMP algo.)
4.Manachar's Algorithm (Palindrome-Searching-Algo)
=>To find the longest palindromic substring within a given text or string.
